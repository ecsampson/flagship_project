# flagship_project


# Project Vision and Goals
  This project will be utilized for gathering weather data to display patterns and anomalies in recent weather events to better analysis trends in extreme weather conditions. Utilizing NOAA APIs in order to gather the data and Python for transforming and cleaning the data and storing in the cloud and Power BI for analytics.

## Primary Outcome
  To better my understanding of data engineering, archetecture and showcasing AI-assisted coding while building a portfolio worthy project.

## Key Skills
- Data Engineering
- Python development
- Understanding of data fundamentals (such as ETL, Batch processing...)
- Utilizing AI as a tool and not just copy and pasting from AI
- Learning and developing my own understanding while working on a project

# Project Scope

### Essential features
- NOAA API data extraction
- Transformation/cleaning the data utilzing Python
- Storing the data in a cloud based server
- Displaying analytics/trends in Power BI

### Non-essential features
- Frontend customer display of the data
- Stream processing
- Multi-lingual capabilities
